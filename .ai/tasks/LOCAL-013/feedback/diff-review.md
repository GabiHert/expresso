# Diff Feedback

Use this file to add review comments on code changes. The `/address-feedback` command will read this file and present your feedback to the agent.

## Format

Use markdown headers to reference specific locations:

- `### path/to/file.ts:42` - Comment on specific line
- `### path/to/file.ts:15-20` - Comment on line range
- `### path/to/file.ts` - General comment about file
- `### General` - Overall task feedback

Mark resolved items with ~~strikethrough~~ or simply delete them.
